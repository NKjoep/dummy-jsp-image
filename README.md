	Usage: 
		place this jsp in your webapp, then point your browser to:
		java-img.jsp?img=/<WIDTH>x<HEIGHT>/<HEX-BACKGROUND>/<HEX-FOREGROUND>&text=

		Example:
		java-img.jsp?img=/640x480/000/fff&text=this is a test
